# TCP/UDP Epoll Server (Test Task)

Небольшой сервер на C++, который обрабатывает TCP- и UDP-соединения с использованием `epoll`.  
Проект реализован как тестовое задание: сервер принимает сообщения от клиентов и поддерживает несколько служебных команд.

## Функциональность

- Поддержка TCP-соединений (echo-сервер для обычных сообщений).
- Поддержка UDP-сообщений.
- Неблокирующие сокеты и механизм мультиплексирования ввода-вывода через `epoll`.
- Команды:
  - `/time` — вернуть текущее серверное время.
  - `/stats` — вернуть количество подключений (total/current).
  - `/shutdown` — остановить сервер.

  ## Стек технологий

- C++ (стандарт C++11/14).
- POSIX sockets (`socket`, `bind`, `listen`, `accept`, `recv`, `send`, `recvfrom`, `sendto`).
- `epoll` для обработки нескольких файловых дескрипторов.
- Компиляция под Linux.

## Структура проекта

- `Server.h` — объявление класса `Server`.
- `Server.cpp` — реализация логики сервера (сокеты, epoll, обработка команд).
- `main.cpp` — точка входа, запуск сервера.

## Сборка и запуск

docker run -it --rm -p 127.0.0.1:12345:12345 my_server
